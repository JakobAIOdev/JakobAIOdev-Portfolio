How to Use This Site

Welcome to JakobAIO's portfolio site! Here's how you can navigate and use the site:

  • Terminal Commands:
    Type commands in the terminal to navigate. For example, type "cd about" to view the about section.
  • Easy Mode:
    Click the "Easy Mode" button to disable terminal input and navigate using clicks.
  • Sections:
    Click on "about", "projects", or "contact" to view the respective sections.
  • Dark Mode:
    Use the toggle switch in the top right corner to switch between light and dark modes.
